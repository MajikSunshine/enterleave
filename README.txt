enterleave
----------

prints a notification when a character enters or leaves the sector

usage:

the script can be configured via two commands

/enterleave_notify_mode <num>, tells the script on what to trigger
0 = off
1 = players (default)
2 = everything

/enterleave_sound <num>, play a soundeffect when a character entered or left
0 = off
1 = entered (default)
2 = left
3 = both
